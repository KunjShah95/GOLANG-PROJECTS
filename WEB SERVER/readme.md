## Getting Started

To get started with this project, follow the instructions below:

1. **Clone the repository:**
   ```sh
   git clone <repository-url>
   ```
2. **Navigate to the project directory:**
   ```sh
   cd /d:/PROJECTS/GOLANG/WEB SERVER
   ```
3. **Build the project:**
   ```sh
   go build
   ```
4. **Run the server:**
   ```sh
   ./webserver
   ```
